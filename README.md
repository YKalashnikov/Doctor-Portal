# Doctor Portal


### Using
- Handlebars 
- Morgan
- Method-override
- Express
- Bootstrap
- Passport
- dotenv
- passport-google-oauth20
- express-session
- nodemon


```shell
$ npm install 
$ npm run dev
$ go to http://localhost:3000
```
```shell
  Add config.env to config folder
  In the file add PORT, MONGO_URI, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET
```
